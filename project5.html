Weather App

A simple weather application that displays temperature, humidity, and changes the background image based on the current weather conditions. Built using HTML, CSS, and JavaScript with API integration.

Project Features

  • Displays Temperature, Humidity, and Weather Conditions
  • Changes Background Image Based on Weather
  • Uses API Integration for Live Weather Data

Technologies Used

  • HTML5, CSS3
  • JavaScript (Fetch API)
  • OpenWeather API

Future Enhancements

  • Hourly and Weekly Forecast: Displaying extended weather data.
  • Geolocation Support: Automatically detect user location for weather updates.
  • Theme Customization: Adding dark and light mode options.
