Fake Hacking Terminal

This project simulates a hacking terminal with animated text effects and fake hacking commands. It was created using HTML, CSS, and JavaScript, leveraging async/await for realistic text delays.

Project Features

  • Realistic Terminal UI
  • Animated Typing Effect
  • Asynchronous Execution with Async/Await

Technologies Used

  • HTML5, CSS3
  • JavaScript (Async/Await)

Future Enhancements

  • Customizable Commands: Adding user-defined commands for interaction.
  • Sound Effects: Incorporating typewriter-style sound effects.
  • Interactive UI: Making the terminal more interactive with user input handling.
