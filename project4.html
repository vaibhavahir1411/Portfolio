Currency Converter

This project is a simple yet functional currency converter that allows users to convert between different currencies using real-time exchange rates. Built using HTML, CSS, and JavaScript, it demonstrates API integration and user-friendly design.

Project Features

  • Real-time currency conversion
  • Easy-to-use UI
  • API integration for exchange rates

Technologies Used

  • HTML5, CSS3
  • JavaScript
  • Exchange Rate API

Future Enhancements

  • Historical Data: Adding a feature to show past exchange rates.
  • Graphical Representation: Visualizing exchange rate trends.
  • Multiple Currency Conversion: Allowing users to compare multiple currencies at once.
