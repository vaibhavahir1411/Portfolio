Age Calculator App

The Age Calculator App is a simple yet useful tool that calculates your age based on the date of birth you enter. It features a clean and user-friendly interface and instantly shows your age in years, months, and days. This project is great for demonstrating form handling, date calculations, and JavaScript logic.

Project Features

  • Instant and accurate age calculation
  • Responsive and clean UI
  • Shows age in years, months, and days
  • Date Validation: Add checks for invalid or future dates

Technologies Used

  • HTML5
  • CSS3
  • JavaScript

Future Enhancements

  • Theme Switch: Option for dark/light mode
  • Next Birthday Countdown: Show how many days left until next birthday
