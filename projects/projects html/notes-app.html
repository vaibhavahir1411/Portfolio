Notes App

This Notes App is a full-stack web application that allows users to add, edit, and delete notes. Each note includes a title and description, and tasks are managed individually for each user. It supports user-specific session handling, enabling a personalized task management experience. Built with Node.js, Express, and EJS.

Project Features

  • User Authentication
  • Add Tasks with Title and Description
  • Edit and Delete Tasks
  • Task Management Per User

Technologies Used

  • Node.js
  • Express.js
  • EJS
  • HTML/CSS/JavaScript

Future Enhancements

  • Authentication with JWT: Secure user sessions using tokens.
  • Task Categories: Ability to categorize and filter tasks.
  • Search Functionality: Quickly search through tasks.
  • Dark Mode: A dark mode toggle for improved user experience.
