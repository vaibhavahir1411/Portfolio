Tic Tac Toe Game

This is a simple yet interactive Tic Tac Toe game built using HTML, CSS, and JavaScript. The game allows two players to take turns marking X and O on a 3x3 grid, automatically detecting the winner or a draw. It features a clean UI and a reset functionality to replay the game.

Project Features

  • 2-Player Mode (X and O)
  • Win and Draw Detection
  • Game Reset Button
  • Responsive Design

Technologies Used

  • HTML5
  • CSS3
  • JavaScript

Future Enhancements

  • Single Player Mode: Add AI to play against the computer.
  • Scoreboard: Track wins/losses for each player.
  • Animations: Add smooth transitions and winning effects.
